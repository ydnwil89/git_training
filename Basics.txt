git commit - создание нового снимка файлов со свежими изменениями

git branch {branchname} - создание новой ветки

git checkout {branchname} - встать на другую ветку

git checkout -b {branchname} - создать ветку и сразу переключиться

git merge {branchname} - слияние двух веток в новый коммит

git rebase - копирует коммиты и переносит в другое место

HEAD - символическое имя текущего выбранного коммита

git checkout {branchname/HEAD}^ - каретка перемещает на коммит выше от выбранного коммита
~<n> - перемещает на n кол-во коммитов выше

git branch -f main HEAD~3 - ключ -f пригодится для перемещения ветки на определенный коммит

git reset {HEAD}- отмена изменений через перенос ссылки на ветку на более старый коммит, хорошо для отката локальных изменений

git revert {HEAD} - отменяет изменения через создание нового коммита с противоположными изменениями
